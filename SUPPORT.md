# Support

This project is maintained on a best-effort basis. Please use the
channels below rather than opening low-priority issues.

## Questions and ideas

Use [Discussions](https://github.com/kaelys-js/claude-multiacct/discussions)
for questions, ideas, and show-and-tell.

## Bug reports and feature requests

Open a bug or feature request on the
[Issues](https://github.com/kaelys-js/claude-multiacct/issues)
tab. Fill out the template; blank issues are disabled by policy. Include
the `claude-multiacct doctor` output for any runtime bug.

## Security

Report security issues privately per [SECURITY.md](SECURITY.md) — never
via a public issue.
